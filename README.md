## 🌐 Como Publicar

### Opção 1: Hospedagem Gratuita (Recomendado para começar)

#### **Netlify (Mais fácil)**
1. Acesse https://www.netlify.com
2. Clique em "Sign up" e crie uma conta
3. Clique em "New site from Git" ou "Drag and drop"
4. Arraste a pasta `retira-multas-site` para a área de upload
5. Pronto! Seu site está online em segundos

#### **GitHub Pages**
1. Crie um repositório no GitHub
2. Faça upload dos arquivos
3. Vá em Settings → Pages
4. Selecione "Deploy from a branch"
5. Escolha "main" branch
6. Seu site estará em `seu-usuario.github.io/retira-multas-site`

#### **Vercel**
1. Acesse https://vercel.com
2. Clique em "New Project"
3. Importe o repositório do GitHub
4. Deploy automático em segundos

### Opção 2: Hospedagem Paga (Mais profissional)

#### **Hostinger, Bluehost, SiteGround**
1. Contrate um plano de hospedagem
2. Acesse o cPanel/Painel de Controle
3. Vá em "Gerenciador de Arquivos"
4. Crie uma pasta chamada `retira-multas`
5. Faça upload dos 3 arquivos (index.html, style.css, script.js)
6. Acesse seu domínio

#### **Seu Próprio Servidor**
1. Copie os 3 arquivos para o servidor
2. Configure o servidor web (Apache, Nginx)
3. Aponte seu domínio para o servidor

### Opção 3: Domínio Customizado

Para usar um domínio customizado (ex: www.retiramultas.com.br):

1. Compre um domínio em:
   - Registro.br (domínios .br)
   - GoDaddy
   - Namecheap
   - Hostinger

2. Configure o DNS para apontar para sua hospedagem

3. Aguarde a propagação (pode levar até 48h)
