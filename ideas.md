# Ideas — Retira Multas Landing Page

## Abordagem 1: Power Black & Gold

<response>
<text>
**Design Movement:** Corporate Power / Premium Legal Branding
**Core Principles:**
- Contraste extremo preto profundo vs. amarelo/dourado vibrante
- Hierarquia tipográfica agressiva com peso visual forte
- Layout assimétrico com blocos de cor sólida como divisores de seção
- Credibilidade construída por densidade de informação bem organizada

**Color Philosophy:** Preto (#0A0A0A) como base de autoridade e seriedade; amarelo dourado (#F5C518) como acento de urgência e ação. Branco apenas para texto em fundos escuros. Cinza escuro (#1A1A1A) para cards e seções alternadas.

**Layout Paradigm:** Seções alternadas preto/cinza-escuro com faixas amarelas diagonais como separadores. Hero com split-screen: texto à esquerda, imagem à direita com overlay escuro.

**Signature Elements:**
- Faixa diagonal amarela como elemento recorrente de separação
- Números grandes em amarelo para estatísticas (ex: "5 ANOS")
- Ícones minimalistas em amarelo sobre fundo preto

**Interaction Philosophy:** Botões amarelos pulsam sutilmente para chamar atenção. Scroll revela elementos com fade-in da esquerda.

**Animation:** Entrada de texto com slide-up, cards com fade-in escalonado, botão WhatsApp com pulse infinito.

**Typography System:** Bebas Neue ou Barlow Condensed (bold) para headlines; Inter ou Roboto para corpo. Hierarquia: 72px/48px/32px/18px/16px.
</text>

## Abordagem 2: Dark Authority (ESCOLHIDA)

<response>
<text>
**Design Movement:** Dark Premium / High-Stakes Legal Service
**Core Principles:**
- Fundo quase-preto com textura sutil (noise/grain)
- Amarelo saturado apenas em CTAs, ícones e destaques — nunca como fundo de seção
- Tipografia condensada e bold para headlines, regular para corpo
- Espaçamento generoso que transmite confiança e profissionalismo

**Color Philosophy:** #0D0D0D como background principal; #F5C518 como cor de ação e destaque; #1C1C1C para cards; #FFFFFF para texto principal; #A0A0A0 para texto secundário. Gradiente sutil de #0D0D0D para #1A1A1A entre seções.

**Layout Paradigm:** Single-column com max-width 1200px, seções com padding generoso. Hero com imagem de fundo com overlay escuro 70%, texto centralizado com badge de credibilidade acima do headline.

**Signature Elements:**
- Linha horizontal amarela fina antes de títulos de seção
- Cards com borda esquerda amarela (4px) em vez de borda completa
- Badge "5 anos no mercado" com fundo amarelo e texto preto

**Interaction Philosophy:** Hover em cards eleva com sombra amarela sutil. Botões com transição de fundo amarelo → amarelo escuro.

**Animation:** Scroll-triggered fade-in com translateY, stagger em cards, contador animado para estatísticas.

**Typography System:** Montserrat (700/800) para headlines; Roboto (400/500) para corpo. Letras maiúsculas em subtítulos de seção.
</text>

## Abordagem 3: Tactical Strike

<response>
<text>
**Design Movement:** Tactical / Military-Inspired Legal Branding
**Core Principles:**
- Grid estruturado com linhas finas como elemento decorativo
- Amarelo como cor de "alerta" e urgência real
- Tipografia monospace para dados e números
- Sensação de precisão e estratégia técnica

**Color Philosophy:** #111111 base; #FFD700 alerta; #222222 cards; linhas decorativas em #333333. Textos em branco puro para máximo contraste.

**Layout Paradigm:** Grid de 12 colunas com elementos que quebram o grid intencionalmente. Seções com bordas finas amarelas como frames.

**Signature Elements:**
- Linhas de grid decorativas em cinza escuro
- Números de seção em formato "01 / 02 / 03" em amarelo
- Ícones de escudo e documento em estilo outline

**Interaction Philosophy:** Hover revela bordas amarelas em elementos. Transições rápidas e precisas.

**Animation:** Entrada com clip-path reveal, números contam de 0 ao valor final.

**Typography System:** Space Grotesk (700) para headlines; IBM Plex Mono para dados; Inter para corpo.
</text>
