📌 OUTRAS OPÇÕES:

GITHUB PAGES (Grátis):
1. Crie conta em github.com
2. Crie repositório "retira-multas"
3. Faça upload dos arquivos
4. Vá em Settings → Pages → Deploy from branch
5. URL: https://seu-usuario.github.io/retira-multas

VERCEL (Grátis):
1. Acesse vercel.com
2. Conecte seu GitHub
3. Selecione o repositório
4. Deploy automático em segundos

HOSTINGER/BLUEHOST (Pago):
1. Contrate hospedagem
2. Acesse cPanel → Gerenciador de Arquivos
3. Faça upload dos 3 arquivos (index.html, style.css, script.js)
4. Acesse seu domínio
